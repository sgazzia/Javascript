# CinePolis - Plataforma Web de Cine
Este proyecto, **CinePolis**, es la simulación de una plataforma web de venta de entradas y confitería para un complejo de cines. Fue desarrollado como un **Trabajo Final** con el objetivo de integrar y aplicar conocimientos avanzados de **HTML, CSS y JavaScript**. La plataforma incluye carga dinámica de contenido a través de API externas, manejo de estado (carrito) con `localStorage`, y validación de formularios.
